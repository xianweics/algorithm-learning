# 数组 (array)

逻辑结构：线性结构。   
存储结构：顺序存储。     


## 题型 

一维数组 : 一维数组从数据结构角度来看好像没啥好学的，实际上是为了学习一些操作一维数组的套路。比如 ：滑窗、双指针等。 有些一维数组的题目暴力解一般都要 O(n^2)，但一般能用这些套路来优化，以降低时间复杂度。

多维数组 : 一般是二维数组题多一点，出题时常会用于表示矩阵、平面直角坐标系。多层循环嵌套难以避免，所以时间复杂度相对高。
